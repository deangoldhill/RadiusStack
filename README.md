# RadiusStack

A complete, modern, containerized RADIUS authentication stack designed for ease of deployment, security, and extensive API-driven management. RadiusStack bundles FreeRADIUS, MariaDB, and a custom Node.js REST API inside a streamlined Docker Compose environment, paired with a sleek, responsive Web UI.

Whether you're managing Wi-Fi access points, enterprise VPNs, or network hardware logins, RadiusStack provides full control over users, billing plans, profile attributes, and real-time accounting without ever needing to touch the command line.

## Key Features
* **Modern Web Interface:** A beautifully designed frontend for managing NAS clients, users, profiles, and billing plans.
* **Full REST API:** Built-in Swagger-style documentation with live cURL generators for every endpoint.
* **Advanced Security:** Supports Time-Based One-Time Passwords (TOTP) for end users and enforced 2FA for administrators.
* **Intelligent Backups:** Export and safely restore JSON-based database configurations across version upgrades.
* **Automated Certificate Generation:** One-click custom SSL/EAP certificate generation or direct file uploads.
* **Live Analytics:** Monitor active sessions, view authentication logs, and generate PDF accounting reports.
